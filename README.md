# Guess My Number

A simple number guessing game where you try to guess the correct number between 1 and 20.

## How to Play

1. Open `index.html` in your browser
2. Enter a number between 1 and 20
3. Click "Check!" to see if you guessed correctly
4. Try to get the correct number with the fewest attempts possible
5. Click "Again!" to start a new game

## Features

- Random number generation
- Score tracking
- High score system
- Simple and clean UI

## Technologies Used

- HTML
- CSS
- JavaScript
